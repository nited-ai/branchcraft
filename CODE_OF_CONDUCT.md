# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. Please read it.

## In short

- Be kind. Assume good faith.
- Patience by design — branchcraft targets people who don't yet know Git well.
- Disagreements are welcome, personal attacks are not.
- The maintainers will enforce the Contributor Covenant in good faith.

## Reporting

If you experience or witness behavior that violates the Contributor Covenant in any branchcraft space (this repository, its discussions, issues, PRs, or related off-platform communications), report it to the maintainers via:

- Direct message to a maintainer on GitHub, or
- Email to the address listed in the maintainer's GitHub profile.

Reports are handled confidentially. The maintainers will respond within 7 days.

## Scope

This applies to all branchcraft project spaces and to any individual representing the project in public.
